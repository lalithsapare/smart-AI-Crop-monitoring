Smart-AI-Crop-monitoring
# About the Project: 
AI Crop Monitoring DL is a deep learning-based crop health monitoring system designed to detect crop problems from images and vegetation data.

This project focuses on early crop diagnosis and helps identify plant health issues before they become severe. It uses deep learning and crop analysis techniques to monitor tomato and rice crops, detect pests and diseases, analyze nutrient deficiencies, and assess crop stress and vegetation health.

# Modules Included: 
# Tomato Disease Detection: 
Detects common tomato leaf diseases from uploaded images.
Helps identify problems early so corrective action can be taken quickly.

# Rice Disease Detection: 
Classifies rice leaf diseases using image-based deep learning.
Useful for monitoring crop health in rice fields.

# Rice Pest Detectio: 
Detects pest-related damage in rice crops.
Supports early pest control and crop protection.

# Nutrient Deficiency Detection: 
Identifies nutrient deficiency symptoms from crop leaf images.
Helps farmers understand whether the plant lacks essential nutrients.

# NDVI Analysis: 
Uses NDVI-based analysis to evaluate plant health and vegetation vigor.
Helps measure crop condition using red and NIR band information.

# Crop Stress Prediction:
Predicts crop stress conditions from relevant input features.
Useful for identifying water stress, nutrient stress, or environmental stress.

# Project Goal: 
The main goal of this project is to build an AI-based crop monitoring system that can:
Detect diseases early.
Identify pest attacks.
Recognize nutrient deficiency symptoms.
Analyze crop health using NDVI.
Predict crop stress conditions.
